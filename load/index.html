A-Frame Preloader Component

Basic Usage

These examples demonstrate dropping in the preloader component to easily add a loading bar to your scenes.

  • Bootstrap Default

    Example showing a full-screen Bootstrap modal preloader. Blocks the entire view until preloading is complete.

  • Bootstrap Small

    Example showing a small Bootstrap modal preloader with close button. Useful if you want part of the scene to load while the user waits for large assets to download.

Advanced Usage

These examples show how to specify your own preloader modal using Bootstrap and third party CSS libraries for displaying a custom spinner in place of the loading bar.

  • Font Awesome + Bootstrap

    Uses custom preloader modal in HTML with custom animation and continue button. Spinner is provided by Font Awesome.

  • Spinkit + Bootstrap

    Uses custom preloader modal in HTML with custom animation. Spinner is provided by Spinkit.
